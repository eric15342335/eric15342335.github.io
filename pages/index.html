As an AI student, you must use a lot of AI chatbot, right?

Yes, I use a lot of AI chatbot. Here are some of the providers I use and my thoughts on them:

ProviderThoughts
GitHub Copilot Education license (see my blog too!)
Poe.com Lots of AI models to pick from, but the quota runs out fast if you use them a lot.

Grok 2 API free $25 per month during Beta + Open-WebUI (or LibreChat)
Ollama Mainly uncensored llama models for fun.
OpenRouter The web interface is a bit clunky, especially when you zoom out.
DeepSeek chat Trying.

The list is not exhaustive, and I will update it as I try more providers.
